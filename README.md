## 【功能介绍】
**影院管理端：主要功能有，登录注册，影院 管理，影片管理，影厅管理，订单管理，用户管理，权限管理等模块。主要是上架影片，排场次，查看订单等。
用户端：用户端主要功能有：登录注册，首页浏览影片，查看榜单，购买电影票等功能。
整体项目还是比较系统的，适合编程学习，二次开发练手，对初学者来说是一个很好的模仿学习项目。**
